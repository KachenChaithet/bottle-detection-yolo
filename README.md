# Bottle Detection YOLO

โปรเจคนี้เป็นส่วนหนึ่งของการเรียนรู้ด้าน Computer Vision และ AI โดยเน้นการตรวจจับวัตถุประเภทขวด (Bottle) แบบ real-time ด้วย YOLO และ OpenCV

## 📌 จุดประสงค์

* ศึกษาการทำงานของ Object Detection ด้วย YOLO
* ทดลองใช้งานกล้องแบบ real-time
* เรียนรู้การจัดการ environment ด้วย Miniconda

## 🧪 สิ่งที่ได้เรียนรู้

* การสร้างและจัดการ environment ด้วย Miniconda
* การติดตั้งและจัดการ dependencies
* การใช้งาน YOLO สำหรับตรวจจับวัตถุ
* การใช้ OpenCV สำหรับประมวลผลภาพจากกล้อง

## 🚀 ความสามารถ

* ตรวจจับขวดจากกล้องแบบ real-time
* ใช้งาน YOLO เบื้องต้น
* โครงสร้างโปรเจคเข้าใจง่าย เหมาะสำหรับผู้เริ่มต้น

## 🛠️ เทคโนโลยีที่ใช้

* Python
* YOLO (Ultralytics)
* OpenCV
* Miniconda

## 📖 หมายเหตุ

โปรเจคนี้จัดทำขึ้นเพื่อการเรียนรู้และทดลองใช้งานเท่านั้น ในอนาคตอาจพัฒนาให้สามารถตรวจจับวัตถุหลายประเภท หรือเพิ่มประสิทธิภาพของระบบ
